# <div align="center">Описание сайта</div>

Сайт будет представлять из себя сайт-визитку компании. На сайте будет страница об аниматрониках [ "Звезды Bon's Burgers" ], страница представляющая персонал компании [ "Персонал" ], отдельная страница-маркетплейс [ "Товары" ], на которой можно будет приобретать мерч BB, а также продуктовые товары, страница с новостями [ "Новости" ], страница меню [ "Меню" ] а также специальная страница с небольшой историей компании с момента ее основания [ "О нас" ]. В проекте также будет реалитзована поисковая система по страницам в сочетании с возможностью авторизации пользователей.

## Целевая аудитория

Основной аудиторией сайта будут являться полконники веб-сериала The Walten Files, поскольку сам сайт будет имитировать работу существующих сайтов реальных сетей ресторанов. Автор надеется на то, что другие поклонники, подобно ему, с интересом отнесуться к данной задумке.

## Ключевой функционал

К основным особенностям сайта можно отнести реализацию поисковой системы, ее суть будет заключаться в том, что любой зашедший пользователь сможет ввести ключевые слова для быстрого перехода между страницами, а также вводить любой желаемый текст и если соответствие с ним будет найдено на странице товаров или странице персонажей пользователю будет выведен результат.

## Причины выбора проекта

Несколько слов о валидности выбора такого рода проекта.  
Стоит упомянуть, что подобного рода проекты уже существуют в сети. Поколонники Five Nights at Freddy's реализовали свой официльный сайт для Freddy Fazbear Pizzeria, где повествование ведется "от лица комании", авторы сериала The Boys ведут социальные сети своих персонажей, в то время как пользователи стараются "подыгрывать" авторам и отвечают словно они сами являются частью событий проекта.

# <div align="center">Схема данных</div>

### User (Пользователь)

| Поле | Описание |
|------|----------|
| id | Уникальный идентификатор пользователя |
| email | Адрес почты, используется для входа |
| username | Отображаемое имя пользователя |
| password_hash | Захешированный пароль |
| created_at | Дата и время регистрации |

### Product (Товар)

| Поле | Описание |
|------|----------|
| id | Уникальный идентификатор товара |
| name | Название товара (3-100 символов) |
| description | Краткое описание товара |
| price | Цена товара |
| image_url | Ссылка на изображение товара |
| is_food | Тип товара: 0 = мерч, 1 = еда |
| category | Категория товара (например: бургеры, одежда) |
| stock | Количество на складе |

## Ключевые эндпоинты API

### Аутентификация

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация нового пользователя |
| POST | `/api/auth/login` | Вход в аккаунт по email и паролю |
| POST | `/api/auth/logout` | Выход из аккаунта |
| GET | `/api/auth/check-email` | Проверка уникальности email |

### Товары

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/products` | Получить список товаров с фильтрацией |
| GET | `/api/products/:id` | Получить детальную информацию о товаре по ID |
| GET | `/api/products/food` | Получить только товары-еду (is_food = 1) |
| GET | `/api/products/merch` | Получить только мерч (is_food = 0) |
| POST | `/api/products` | Создать новый товар |
| PATCH | `/api/products/:id` | Обновить данные товара |
| DELETE | `/api/products/:id` | Удалить товар |
